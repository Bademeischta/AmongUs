# Changelog

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/).

## [1.0.0] - 2024-08-01

### Hinzugefügt (Added)

- **Initiales Release des Jester Role Mods.**
- **Rollen-Management-System:** Eine robuste und erweiterbare Architektur zur Verwaltung von benutzerdefinierten Rollen.
- **Jester-Rolle:** Implementierung der Jester-Rolle mit der Gewinnbedingung, bei einem Meeting rausgewählt zu werden.
- **Netzwerk-Synchronisation:** Stabile Synchronisation von Rollen und Spieleinstellungen über Custom RPCs.
- **Benutzerdefinierte Spieleinstellungen:** Eine Option in der Lobby, um die Wahrscheinlichkeit für einen Jester einzustellen (0-100%).
- **UI-Anzeigen:** Der Jester sieht seinen Namen in Pink und erhält eine Textanzeige "Rolle: Jester" auf dem HUD.
- **Projekt-Setup:** Eine `.csproj`-Datei für einfaches Kompilieren und eine `README.md` für die Dokumentation.
